Información básica

Esta ventana permite Registrar un usuario nuevo.

Requirimientos de logueo

Para crear un usuario debera introducir sus datos y asi poder habilitar el boton Registrar, en caso de que un campo sea introducido incorrectamente se le notificara el problema en cuestion.
Para el correcto registro se debera introducir un Email correcto con su formato correcto ej: pedro.pedrin@gmail.com.
Nombre de usuario tendrá que ser único, en caso contrario se le notificará y ademas tendra un minimo de 3 caracteres.
Contraseña tendra que tener mínimo 3 caracteres y se tendrá que repetir en confirmar contraseñas y tienen que coincidir. Una vez realizadas las acciones anteriores, deberá pulsar el botón Registrar para crear el usuario. Si no se produce ningún error, se habra dado de alta al usuario y regresará a la ventana de inicio.

Cerrar ventana

Pulsando el botón Cancelar se cerrará la ventana y volvera al início.
